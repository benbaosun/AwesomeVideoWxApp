# AwesomeVideoWxApp
《倾心短视频》微信小程序

### 小程序功能

+ 注册、登陆账号
+ 发布短视频，并添加背景乐
+ 浏览短视频
+ 收藏短视频
+ 评论、转发短视频
+ 关注用户
